Tokens and components extracted from the live GuidingCare application and Figma file. All components reflect actual production UI observed in screenshots.

Color

Teal is the primary action/brand color. Blue is used for links, informational context, and the Digital channel badge. The UI is predominantly white and light gray — teal appears as accents, not large fills.

Typography

Open Sans throughout. The UI is dense — body text is 12px, not 14px. Bold (700) for section headings and table column heads; SemiBold (600) for tab labels and data values; Regular (400) for label text in key-value pairs. Section headings are teal.

Spacing & Border Radius

The GC UI is compact. Most component padding is 6–12px. Row heights are 36px (tables) and 44px (nav bar). Border radii are small — 3–4px for most elements.
